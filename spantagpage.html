<span> Tag Usage and Definiton

  • <span> is an inline container used to mark up a part of a text, or a part of a document.
  • <span> is easily styled by CSS or manipulated with JavaScript using the class or id attribute.
  • <span> is much like the <div> element, but <div> is a block-level element and <div> is an inline element.


Examples
<p> I have <span> style="color:blue" blue </span> eyes.</p>

I have blue eyes.
