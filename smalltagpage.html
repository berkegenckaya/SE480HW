<small> Tag Usage and Definition

The <small> tag defines smaller text (like copyright and other side-comments). Tip: This tag is not deprecated, but it is possible to achieve richer (or the same) effect with CSS.

Examples

This is some normal text.

This is some smaller text.

<p>This is some normal text.</p>
<p><small>This is some smaller text.</small></p>
