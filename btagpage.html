<b> Tag Usage and Definition

The <b> tag specifies bold text without any extra importance. You can also use the following CSS to set bold text: "font-weight: bold;".According to the HTML5 specification, the <b> tag should be used as a LAST resort when no other tag is more appropriate. The specification states that headings should be denoted with the <h1> to <h6> tags, emphasized text should be denoted with the <em> tag, important text should be denoted with the <strong> tag, and marked/highlighted text should be denoted with the <mark> tag.


Examples

This is normal text - and this is bold text.

<p>This is normal text - <span> style="font-weight:bold;">and this is bold text</span>.</p>
