<table> Tag Usage and Definition

The <table> tag defines an HTML table. An HTML table consists of one <table> element and one or more <tr>, the <th>, and the <td>elements. The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell. An HTML table may also include <caption>, <colgroup>, <thead>, <tfoot>, and <tbody> elements.

Examples
<table>
<tr>
<td>Month</td>
<td>Savings</td>
</tr>
<tr>
<td>January</td>
<td>$100</td>
</tr>
</table>
